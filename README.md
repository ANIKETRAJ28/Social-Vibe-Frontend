# Social-Vibe

## Project Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/ANIKETRAJ28/Social-Vibe-Frontend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Social-Vibe-Frontend
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
